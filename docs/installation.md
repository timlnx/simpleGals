## Requirements

- Python >= 3.12
- A terminal emulator with Kitty graphics protocol, iTerm2 inline image protocol,
  or Unicode truecolor support (required for `sgui` image previews)

## Install

```bash
pip install simplegals
```

## Local build from source

```bash
git clone https://github.com/timlnx/simpleGals.git
cd simpleGals
pip install -e ".[dev]"
```
